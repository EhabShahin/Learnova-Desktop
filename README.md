# Learnova Video Helper

Desktop application for playing videos with integrated Pomodoro timer.

## Features
- YouTube & Udemy video playback with Pomodoro timer
- Automatic progress tracking
- Synchronization with Learnova Study Planner
- Udemy Premium support
